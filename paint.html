Shortcuts
w / s
- Cycle colors
c
Make a new canvas, size parameters are defined as <width>, <height>
q
Show grid lines
e
Erase grid
shift/RMB
Draw freely with the cursor (shift to help trackpad users)
z/x or -/+
Increase/decrease pixel size

d
Add a custom color

There are 11 colors, the 11th color is a HTML5 color picker that you can select any color you prefer with.
Tip: make sure you properly specify values when making a new drawing grid
